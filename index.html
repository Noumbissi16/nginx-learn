About Me

I'm Noumbissi Stael, a software developer currently learning DevOps, Cloud Infrastructure and Distributed Systems. This project is a simple Node.js application that serves a static HTML file. The objective is to pratice containerization and deployment using Docker and nginx. Load balancing and automatic container scaling . This align with my DevOps learning path and my goal is to build scalable systems and improve my DevOps skills through real-world projects.

Run the Application

Run locally

Install dependencies: npm install

Start the server: npm start


Run with Docker

Build image: docker compose up --build --scale app1=5 5 precise the number of containers to run

Access the application at: http://localhost
